# Vídeo de Apresentação

## Requisitos

- **Duração máxima**: 5 minutos
- **Formato**: MP4 ou formato compatível
- **Conteúdo**: Demonstração da solução em funcionamento

## Roteiro Sugerido

### 1. Introdução (30 segundos)

- Apresentação do projeto
- Objetivo do sistema
- Tecnologias utilizadas

### 2. Demonstração - Navegação (1 minuto)

- Home page
- Navegação por categorias
- Visualização de produtos
- Adicionar ao carrinho

### 3. Demonstração - Cadastro e Login (1 minuto)

- Criar conta
- Fazer login
- Cadastrar endereço

### 4. Demonstração - Compra (1,5 minutos)

- Ver carrinho
- Finalizar compra
- Seleção de loja
- Confirmação de pedido
- Visualizar pedidos

### 5. Funcionalidades Técnicas (1 minuto)

- API funcionando
- Banco de dados
- Integrações

### 6. Conclusão (30 segundos)

- Resumo do que foi desenvolvido
- Próximos passos
- Agradecimentos

## Checklist de Gravação

- [ ] Áudio claro e sem ruídos
- [ ] Tela em boa qualidade (1080p mínimo)
- [ ] Navegação fluida sem travamentos
- [ ] Todas as funcionalidades principais demonstradas
- [ ] Duração dentro do limite (5 minutos)
- [ ] Vídeo editado e finalizado

## Dicas para Gravação

1. **Prepare o ambiente**:
   - Feche aplicações desnecessárias
   - Use navegador em modo anônimo (dados limpos)
   - Tenha dados de teste prontos

2. **Qualidade do vídeo**:
   - Use ferramenta de gravação de tela (OBS, Loom, etc.)
   - Grave em resolução alta (1080p ou superior)
   - Frame rate: 30fps ou superior

3. **Narração**:
   - Fale claramente
   - Explique o que está fazendo
   - Evite pausas longas
   - Pratique antes de gravar

4. **Edição**:
   - Remova pausas desnecessárias
   - Adicione transições se necessário
   - Adicione legendas se útil
   - Verifique áudio e vídeo sincronizados

## Ferramentas Recomendadas

### Gravação de Tela
- **OBS Studio** (gratuito, open source)
- **Loom** (gratuito, fácil de usar)
- **ScreenFlow** (Mac, pago)
- **Camtasia** (Windows/Mac, pago)

### Edição
- **DaVinci Resolve** (gratuito, profissional)
- **Shotcut** (gratuito, open source)
- **Adobe Premiere** (pago, profissional)
- **iMovie** (Mac, gratuito)

## Exemplo de Script

```
[00:00-00:30] Introdução
"Olá! Este é o projeto Cupcakes CS, um e-commerce de cupcakes 
desenvolvido para o PIT II. O sistema foi construído com Next.js 
no frontend e NestJS no backend, utilizando PostgreSQL como banco 
de dados."

[00:30-01:30] Navegação
"Agora vou demonstrar a navegação. Na home page temos as categorias 
e produtos em destaque. Ao clicar em uma categoria, vemos os cupcakes 
disponíveis. Posso adicionar produtos ao carrinho facilmente."

[01:30-02:30] Cadastro
"Para fazer uma compra, preciso criar uma conta. Vou preencher os 
dados e criar a conta. Agora vou fazer login e cadastrar um endereço 
de entrega."

[02:30-04:00] Compra
"Com a conta criada e endereço cadastrado, vou ao carrinho e finalizo 
a compra. O sistema busca automaticamente lojas que atendem minha região. 
Após confirmar, o pedido é criado e posso visualizá-lo na área de pedidos."

[04:00-05:00] Conclusão
"O sistema está funcional e atende aos requisitos do projeto. 
Todas as funcionalidades principais foram implementadas e testadas. 
Obrigado pela atenção!"
```
